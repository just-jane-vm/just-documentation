# overview
it is possible to whisper [jill](www.twitch.tv/just__jillian) to give her information about yourself.
jill will have access to this information when responding to you in stream.

## update preferred name
use the `!call-me` command when whispering jill to update the name she will use to refer to you.

### example

```
[user]: /w just__jillian !call-me whenever
[just__jillian]: sure thing whenever!
```

## update about-me
use the `!about-me` command to add information about yourself to things that jill knows.
you can clear out this information at any time.
jill may refer to this during stream please do not say anything you would prefer to be private.

### example

```
[myk0ll]: /w just__jillian !about-me i am doing to debut as a foxgirl vtuber soon.
```

## remove about-me
use the `!forget-me` command to remove all data that jill has stored about you.

### example

```
[user]: /w just__jillian !forget-me
[just__jillian]: forget who?
```
